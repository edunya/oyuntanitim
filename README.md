# OyunTanıtım

OyunTanıtım, sevilen ve güncel oyunlar hakkında özet bilgiler, incelemeler ve bağlantılar sunan basit bir web sitesidir.

## Neler Var?

- **Ana Sayfa (`index.html`)**
  - Öne çıkan oyunlar
  - Son duyurular ve kampanyalar
  - Kısa tanıtım metinleri

- **Oyun Sayfaları**
  - `games.html` ve `game-*.html` dosyalarında seçili oyunlar için:
    - Oyun türü
    - Oynanış notları
    - Sistem gereksinimleri
    - Platform bağlantıları (Steam vb.)

- **Diğer Sayfalar**
  - `duyurular.html` – oyun duyuruları ve etkinlikler
  - `istatistikler.html` – sitede yer alan oyunlara dair temel istatistikler
  - `hakkinda.html` – proje ve site hakkında genel bilgiler
  - `iletisim.html` – iletişim formu

- **Admin Arayüzleri**
  - `admin-duyurular.html`, `admin-oyunlar.html`, `admin-birimler.html`
  - Yalnızca **örnek yönetim arayüzü** amaçlıdır, gerçek veri kaydı yapılmaz.

## Kullanılan Teknolojiler

- HTML5, CSS3
- Bootstrap ve temel JavaScript eklentileri

## Çalıştırma

1. Proje klasörünü bilgisayarına kopyalayın.
2. `index.html` dosyasını bir web tarayıcısında açın.
3. Menü üzerinden veya ilgili dosyalara tıklayarak diğer sayfalara geçiş yapın.
